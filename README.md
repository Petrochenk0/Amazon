# Amazon Clone on React / TypeScript / Nest / TailwindCSS / PostgreSQL 🛒🔥

**Здарова!** Добро пожаловать в мой **AMAZON-STORE**, тут не просто скучный магазин, а **НАСТОЯЩАЯ БОМБА** с крутыми фичами и дизайном, который **рвёт все шаблоны!** 🔥 Построено на **React**, **Nest.js**, **TypeScript**, **PostgreSQL** и **TailwindCSS** — для **максимально охуенного** опыта пользователей! 😏

## Основные Фичи 💥

- **🔍 Поиск по товарам** — быстро найдёшь что угодно, даже если сам не знаешь, что ищешь! 😎  
- **📦 Просмотр отдельных товаров** — хочешь увидеть все характеристики? Да легко! Тут все, что нужно, видно!  
- **🛒 Добавление в корзину** — любой товар можно **вжух** и добавить! Больше не надо бегать за продуктами в магаз, всё тут!  
- **🔐 Регистрация и Авторизация** — всё как надо! Зарегистрировался и наслаждаешься шопингом! 🤘  
- **🔄 Сортировка** — выбирай по цене, популярности или по тому, что тебе в кайф! Всё под твоим контролем, братан! 🔥

## Технологии, которые тут РУЛЯТ! 🚀

- **React**: Передача данных молниеносная! Переходишь по товарам как по маслу! ⚡
- **Nest.js**: Сервер работает как танк, надёжный и непробиваемый! Никаких задержек! 🛡️
- **TypeScript**: Чистота кода и никаких багов, всё чётко по типам, как и должно быть! 💯
- **PostgreSQL**: База данных хранит всё, от пользователей до товаров! Вся инфа надёжно под капотом! 🗃️
- **TailwindCSS**: Визуал настолько стильный, что каждый раз захочешь зайти просто полюбоваться! 👁️‍🗨️
